# Summer School Program 2026 — Project Guide

## Overview

Welcome to the Microsoft Summer School 2026! This document describes the available projects, prerequisites, and program structure.

## Available Projects

### 1. Building Your First Local RAG Application with Foundry Local
Learn how to build a retrieval-augmented generation application that runs entirely on your device using the Foundry Local SDK. You will work with local LLM models, embedding models, and vector search.

Prerequisites: Python 3.11+, basic knowledge of NLP concepts.

### 2. Stock Price Prediction with PyTorch
Build a machine learning model to predict stock prices using PyTorch. Covers data preprocessing, model architecture, training, and evaluation.

Prerequisites: Python, basic statistics, pandas.

### 3. Quantum Kickstart with Q#
Get started with quantum computing using Microsoft's Q# language. Learn quantum gates, circuits, and run your first quantum program.

Prerequisites: Basic linear algebra, interest in quantum computing.

### 4. Titanic Survival Analysis (High School Students Only)
Use Python and pandas to analyze the Titanic dataset. Learn data cleaning, visualization, and basic machine learning.

Prerequisites: Basic Python, willingness to learn.

## Program Schedule

- Kickoff Meeting: June 29, 2026
- Duration: 4 weeks
- Final Presentations: July 27, 2026
- Format: Fully remote
